# 실시간 객체 모자이크 (Mosaic)

웹캠 영상에서 얼굴을 실시간으로 감지해 모자이크 처리하는 프로그램입니다.
Haar Cascade로 얼굴을 탐지하고, 눈·코·입·귀 검출로 오탐을 걸러낸 뒤
픽셀화 처리로 가립니다. 이미지 파일과 실시간 웹캠 두 가지 모드를 지원합니다.

## 사용 기술
C++, OpenCV

## 주요 기능

**얼굴 탐지 + 이목구비 기반 오탐 제거**
정면/측면 얼굴을 Haar Cascade로 탐지한 뒤, 얼굴 영역 안에서
눈·코·입·좌우 귀를 추가로 검출합니다. 이목구비가 2개 이상 잡힐 때만
실제 얼굴로 인정해, 배경의 오탐을 줄였습니다.

**놓친 얼굴 추적 (TrackedFace)**
탐지된 얼굴을 framesTracked 카운트로 관리해, 일시적으로 탐지에
실패한 프레임에서도 모자이크가 유지되도록 했습니다.
마지막으로 본 지 3초가 지난 얼굴은 추적 목록에서 제거합니다.

**std::async 기반 비동기 처리**
메인 루프는 프레임 캡처와 화면 출력만 담당하고, 무거운 모자이크
연산은 std::async로 백그라운드에서 처리합니다. 이전 연산이 끝났을
때만 다음 프레임을 넘겨, 화면이 끊기지 않도록 했습니다.
공유되는 추적 맵은 mutex로 보호했습니다.

## 모드
1. 이미지 파일에서 얼굴 모자이크 처리
2. 실시간 웹캠 모자이크 처리 (ESC로 종료)

## 실행 방법
Haar Cascade xml 파일 경로가 필요합니다. CascadePaths.h에 정의된
경로에 OpenCV의 cascade 파일이 있어야 합니다.
